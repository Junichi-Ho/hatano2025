ToCSVシートをCSVファイルとして出力するVBAコードの使用方法

【ファイル説明】
1. VBA_ToCSV_Export.bas - 基本的なCSV出力コード
2. VBA_Complete_ToCSV_Export.bas - 詳細で実用的なCSV出力コード

【使用方法】

1. Excel VBAエディタを開く
   - Alt + F11 を押す
   - または「開発」タブ → 「Visual Basic」をクリック

2. 新しいモジュールを挿入
   - 「挿入」 → 「モジュール」をクリック

3. コードをコピー＆ペースト
   - 提供されたコードをモジュールに貼り付け

4. 実行方法
   - F5キーを押す
   - または「実行」 → 「Sub/UserFormの実行」をクリック

【利用可能な関数】

1. ExportToCSV()
   - ToCSVシートを「Hatanoscore.csv」として出力
   - 既存ファイルがある場合は上書き

2. ExportToCSVWithDate()
   - 日付を含むファイル名で出力
   - 例：「Hatanoscore_20241201.csv」

3. CopyToCSVAndExport()
   - 既存のコピーペースト処理に追加する例
   - 既存コードの最後に「Call ExportToCSV」を追加

【既存コードへの追加方法】

既存のコピーペーストコードの最後に以下を追加：

```
' CSVファイルとして出力
Call ExportToCSV
```

例：
```
Sheets("ToCSV").Select
Range("A1").Select
Selection.Copy
' 既存のコピーペースト処理...

' CSVファイルとして出力
Call ExportToCSV
```

【注意事項】

1. ファイルはExcelファイルと同じフォルダに保存されます
2. 既存の「Hatanoscore.csv」がある場合は上書きされます
3. エラーが発生した場合はメッセージボックスで通知されます
4. 日付形式は「yyyy/m/d」形式で出力されます

【トラブルシューティング】

1. 「ToCSV」シートが見つからない場合
   - シート名が正確に「ToCSV」であることを確認
   - 大文字小文字は区別されます

2. ファイルが保存できない場合
   - ファイルが他のアプリケーションで開かれていないか確認
   - フォルダの書き込み権限を確認

3. 文字化けが発生する場合
   - Excelの文字コード設定を確認
   - 必要に応じてコードの文字エンコーディングを調整
